## Instructions

Have the AI agent update this log at the end of each work session. Each entry
captures what happened, what's next, and anything blocking progress. This log
serves as the primary handoff document so the next session's AI agent can
restore context.

**When to update:**

- At the end of every work session (AI agent writes the entry)
- When handing off to another engineer
- When a blocker is encountered or resolved

**How AI uses this log:** At the start of a session, the AI agent reads this log
to restore context and suggest priorities. At the end of a session, the AI agent
writes a session entry summarizing what was accomplished.

> **AI agent instruction:** _"Before ending this session, update the session log
> with what was completed, what's in progress, next steps, and any blockers."_
